Classcentral: Usability Testing
Year ✦ 2023
Usability Testing on an MOOC website
Description

Class Central is a leading platform for online learning, offering courses from top universities with an easy-to-use interface and powerful search tools.

We conducted interviews with 8 users to gather key insights, which guided the redesign of several aspects of the website. The focus was on improving usability, enhancing the overall user experience, and aligning the design with user needs and behaviors for a more intuitive interface.
